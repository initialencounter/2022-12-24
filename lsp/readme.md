# koishi-plugin-lsp

[![npm](https://img.shields.io/npm/v/koishi-plugin-lsp?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-lsp)

抄袭自卖阔落的[crossdressing-detect](https://github.com/MicroCBer/dressing-detect-server)



# 后端搭建

* 安装依赖

```
pip install -r requirements.txt
```
* [下载模型](https://github.com/MicroCBer/dressing-detect-server/releases/tag/0.0.1)

* 启动服务

>有显卡启动:
```python main_cuda_1st.py```
或者```python main_cuda_2ed.py```

>无显卡启动:
```python main_cpu.py```
