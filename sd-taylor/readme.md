# koishi-plugin-sd-taylor

[![npm](https://img.shields.io/npm/v/koishi-plugin-sd-taylor?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-sd-taylor)


AI绘图插件,支持controlnet,lora,AI-Tag增强


## 目录结构
### 已经做好了的
- [x] 文字绘图
- [x] 以图绘图
- [x] 图片超分辨率
- [x] 识图
- [x] 切换模型
- [x] lora查看
- [x] controlnet


### 指令如下：
| 功能 | 指令 |
|  ----  | ----  |
| 文字绘图 | tl 描述 |
| 以图绘图 | tl.img 原图 描述？ |
| 超分辨率 | tl.ext 原图 |
| 识图 | tl.txt 图片 |
| 切换模型 | tl.switch|
| lora查看 | tl.lora |
